# Cryto Trakers
가상화폐 API를 이용한 프로젝트

https://reasonjeans.github.io/cryto_traker


## using
- Typescript

- Styled Components

- React Query

- React Router DOM

- Recoil

- Apex Charts


## 기능
코인 api 렌더링

코인 리스트, 상세 화면

Apex Charts 라이브러리를 이용한 차트 구현

다크모드, 라이트모드 구현


## 화면
- 코인 리스트 화면
  - 코인 100가지 리스트

- 코인 상세 화면
  - 티커, 순위, 현재가, 총량, 최대 발행량

- 가격

- 차트
  - 라인, 캔들스틱 차트


## 설치 및 실행
설치
```
$ npm i
```

실행
```
$ npm start
```
